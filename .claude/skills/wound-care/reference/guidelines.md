# Guideline reference (condensed)

Source standards the wound-care skill grounds recommendations in. Cite the named
standard in parentheses; this is a working summary, not a substitute for the
primary documents or clinician judgment.

## NPIAP 2019 — Pressure injuries
- Staging 1–4, Unstageable, Deep Tissue Injury; do **not** reverse/back-stage.
- Assessment: location, size, depth, tissue type, exudate, undermining/tunneling,
  edges, periwound, pain.
- Prevention bundle: repositioning (≈q2h), pressure-redistribution support surface,
  heel offloading, prophylactic dressings over bony prominences.
- Nutrition: protein 1.25–1.5 g/kg/day, 30–35 kcal/kg/day; dietitian if at risk.

## IWGDF / IDSA 2023 — Diabetic foot
- Offloading is first-line: total-contact cast / irremovable knee-high device.
- Classify (SINBAD / WIfI / University of Texas / Wagner).
- Infection severity (IWGDF/IDSA): mild / moderate / severe → escalate workup & abx.
- Osteomyelitis: probe-to-bone + plain film, then MRI; bone culture; targeted abx.
- Perfusion: assess ABI / toe pressures; revascularize ischemic limbs.

## WOCN — Lower-extremity venous/arterial
- **ABI before compression.** ABI ≥ 0.8 → graduated compression. ABI < 0.8 →
  avoid/modify compression and refer vascular; < 0.5 = critical limb ischemia (urgent).
- Venous ulcers: compression + elevation are the mainstays.

## TIME — Wound-bed preparation
- **T**issue (debride non-viable), **I**nfection/inflammation (bioburden control),
  **M**oisture balance (exudate-matched dressings), **E**dge (advancement;
  reassess if edges not migrating → consider advanced therapy).

## Infection signs — NERDS / STONEES
- NERDS (superficial/local): Non-healing, Exudate, Red friable tissue, Debris, Smell
  → topical antimicrobials.
- STONEES (deep/spreading): Size↑, Temperature↑, Os (probe-to-bone), New breakdown,
  Erythema/Edema, Exudate, Smell → systemic antibiotics + workup.

## Healing trajectory
- < 40% wound-area reduction by week 4 predicts non-healing → reassess etiology,
  offloading, perfusion, nutrition; consider CTP/skin substitute or NPWT.

## ISTAP — Skin tears
- Type 1 (no tissue loss) / 2 (partial flap loss) / 3 (total flap loss).
- Approximate flap, atraumatic silicone dressing, mark removal direction; prevention
  (emollients, limb protection).
